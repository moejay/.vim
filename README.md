# My .vim stuff

## Installation - Tested on vim 7.4

### Clone the repo

`$ cd && git clone ssh://git@github.com/moejay/.vim`

### Get vim plugins

`$ cd .vim && git submodule init && git submodule update`

### Move to home folder
`$ mv .vimrc ../`


## Plugin manager
Plugins managed by pathogen
