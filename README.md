# My .vim stuff

## Installation - Tested on vim 7.4

### Clone the repo

`$ cd && git clone ssh://git@github.com/moejay/.vim`

### Move the .vimrc file

`$ mv ~/.vim/.vimrc ~/.vimrc`

## Plugin manager
Plugins managed by vundle

open vim and run `:PluginInstall` 


## Plugins included
* Nerdtree
* vim-sensible
* And more
